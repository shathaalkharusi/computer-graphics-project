Example of Bresenham algorithm



Using Bresenham algorithm, generate the coordinates of the pixels that lie on line segment having the end point (30,18) and (20,10):


Step1: First off calculate all of the constants

  1. Δx: 10 --> 30-20
  2. Δy: 8 --> 18-10
  3. 2Δx: 20 --> 2*10
  4. 2Δy: 16 --> 2*16
  5. 2Δy - 2Δx: -4

Step2: Calculate the initial decision parameter p0:

  1. p0 = 2Δy – Δx = 6

Step3: At each Xk along the line, starting at k = 0, perform the following test.

  1. If (Pk < 0), the next point to plot is (Xk+1, Yk) and Pk+1= Pk + 2Δy.
  2. Otherwise,the next point to plot is (Xk+1, Yk+1) and:Pk+1= Pk + 2Δy - 2ΔX

Step4: Repeat step 3 (Δx – 1) times.

p0 Pk (Xk+1, Yk+1)
0 2Δy - ΔX = 6 > 0 (21, 11)
1 6 + -4 = 2 > 0 (22, 12)
2 2 + -4 = -2 < 0 (23, 12)
3 -2 + 16 = 14 > 0 (24, 13)
4 14 + -4 = 10 > 0 (25, 14)
5 10 + -4 = 6 > 0 (26, 15)
6 6 + -4 = 2 > 0 (27, 16)
7 2 + -4 = -2 > 0 (28, 16)
8 -2 + 16 = 14 < 0 (29, 17)
9 14 + -4 = 10 > 0 (30, 18)

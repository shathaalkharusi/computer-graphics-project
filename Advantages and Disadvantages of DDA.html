Advantages Disadvantages
This approach is quicker than using a line equation directly. Rounding off and floating point additions are involved. Error accumulation results from round-off error accumulation.
The multiplication theorem is not used in this procedure. It takes a long time to do floating point and rounding off procedures.
Because it enables us to track changes in x and y values, it is not feasible to depict the same spot twice. Using the software to generate lines is more appropriate. It is less appropriate for hardware implementation, though.
When a point is moved, this procedure indicates an overflow.
Because there are only two additions required in each stage, the approach is simple.
